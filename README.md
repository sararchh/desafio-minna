## 📝 Scripts

Instale as dependências.

  

```bash
$ yarn
```

  

Para executar o projeto rode o comando abaixo.

```bash
$ yarn start
```
ou
```bash
$ npm run start
```
  
  

Para realizar o build basta executar o comando abaixo que com o package do next será feito o build.

  

```bash
$ yarn build
```
